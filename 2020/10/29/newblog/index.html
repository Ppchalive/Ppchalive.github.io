今天学习了Hexo发布微博

hexo基本使用命令

hexo init blog
创建博客文件夹

npm install hexo-deployer-git --save
安装发布工具在博客文件夹内

  1. hexo clean 清除缓存，快速生效博客
  2. hexo generate 生成静态文件
  3. hexo deploy 把生成文件部署到博客上

创建新博客

hexo new blogName
创建博客文件
